	************************************
*****  	    Description de l'application:   	*****
	************************************
ChooseYourCharacter est un site web codé à partir de JSF. Il permet à l'internaute de se créer un personnage en entrant son nom, son prénom, son âge, et en choisissant dans un menu son pays d'origine.

Une fois ces choix validés en cliquant sur le bouton "Submit" ou "Valider" en version française, l'internaute peut retrouver le personnage qu'il a créé.

L'application comporte une version anglaise, et une version française en cours de finalisation.

Des messages d'erreur sont paramétrés pour réguler le remplissage des champs du formulaire.

Pour afficher le site : 
Télécharger le projet depuis github, avec l'url https://github.com/PHLegourd/JSFVanPH.git
Importer le projet dans Eclipse EE.
Clic droit sur le projet, sélectionner Run on server.
